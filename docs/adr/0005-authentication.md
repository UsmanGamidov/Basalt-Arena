# ADR 0005. Аутентификация: JWT + отзыв через БД, argon2

- **Статус:** принято
- **Дата:** 2026-06

## Контекст

Нужны безопасные сессии с возможностью отзыва (logout, смена прав) на одном
инстансе приложения, без внешнего стора (см. ADR 0002).

## Решение

- **Access-токен** — JWT (HS256, 15 мин) с `jti`; `jti` хранится в `AuthToken`.
  Проверка запроса сверяет `jti` с БД — это даёт отзыв (logout удаляет запись).
- **Refresh-токен** — opaque UUID в `AuthToken` (30 дней), ротируется при refresh.
- **Пароли** — argon2id для новых хэшей; проверка обратно совместима с bcrypt и
  legacy plaintext, который однократно апгрейдится при старте.
- **Rate-limit** — глобальный throttler + жёсткие лимиты на login/register/refresh.

## Последствия

- Отзыв сессий без Redis; на одном инстансе сверка `jti` с БД дёшева.
- При масштабировании сверку `jti` можно вынести в Redis (точка пересмотра).
